# Prompts 폴더

이 폴더에는 AI 어시스턴트 프롬프트가 저장됩니다.

## 사용 방법

### 로컬 개발
1. `system-prompt.txt` 파일 수정
2. `.env` 파일에 복사-붙여넣기
3. 개발 서버 재시작

### Vercel 배포
1. `system-prompt.txt` 파일 수정
2. Vercel Environment Variables에서 `SYSTEM_PROMPT` 수정
3. 재배포

## 파일 설명

- `system-prompt.txt`: 기본 시스템 프롬프트
- GitHub에 올라가도 됩니다 (민감한 정보 없음)
